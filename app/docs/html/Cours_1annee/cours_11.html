Cours 11

Adjectifs en い

大おおきい = grand

Forme négative : 大きい(く) + ない　＝　大きくない = pas grand

あたらい　= nouveau

Forme négative : あたらくな = pas nouveau

Cas particulier : いいです (c’est bon) ==> よくないです (c’est pas bon)

Adjectifs en な

べんりな　(便利な) = pratique

Forme négative : べんりな（では　ou　じゃ）　+ ない = べんりではない ou べんりじゃない　= pas pratique

Vocabulaire

  • 電子でんしレンジ = microwave
  • 電子でんし = électron
  • いす = tabouret / chaise
  • にぎゃかな = animé
