Nom + のようだ : On dirait + Nom

例１：「冬(ふゆ)」　今日は　あたたかくて，春(はる)のようだ。

例２：「秋(あき)」　今日は　さむくて、冬のようです。

例１：「秋」　今日は、春のようにあたたかい。
