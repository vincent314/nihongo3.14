Écrire un document LaTex en japonais

LaTex est une façon très élégante de rédiger des documents, pour laquelle de nombreuses extensions existent afin de supporter un panel très large d’exigences dans le domaine de la publication. Je vais vous parler ici de la spécificité pour écrire en japonais dans un document LaTex.

Les outils à installer pour arriver à notre but sont :

  • TexLive : LaTex packagé, la manière la plus facile pour commencer. Peut comporter tout ou une partie des outils suivants :
  • XeTex : un remplacement de pdfLatex avec le support UNICODE, indispensable pour la rédaction de documents mélangeant français/japonais
  • Font vlgothic : une police de caractères asiatiques.
  • Gummi : un outil d’édition avec compilation et aperçu à la volée.

Donc, pour résumer, l’installation sous Linux Ubuntu/Mint :

sudo apt-get install texlive texlive-xetex latex-cjk-xcjk fonts-vlgothic gummi

Dans les préférences de Gummi, pensez à spécifier la compilation avec XeTex.

Gros bémol toutefois avec cette technique : nous perdons la mise en forme en italique et en gras.
