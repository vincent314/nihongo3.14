Leçon 3 du livre

  • そうですね = en effet …
  • 食たべる = manger (cf. 良いい = bon)
  • …ましょう = proposition. Ex : たべましょう = mangeons !

Accoler 2 adjectifs

  • きれい で たかそうです。 （きれいな　＋　たかい） = ça a l’air joli et cher.

  • 1000円えん です。==> 1000円でした。
  • いくらですか。 ==> いくらでした

Pour les adjectifs, on utilise かった :

  • たかいです==>たかかったです。
  • やすいです==>やすかったです。
