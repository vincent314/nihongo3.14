Spécialités (せんもん)

  1. あなた　の　せんもん　は　なんですか　？　Quelle est ta spécialité ?
  2. おがく　の　せんせい　= professeur de musique
  3. リール　の　しょうぼっし = pompier de Lille
  4. 山田やまださん　は　日本　の　ぎんこういん　です。= M. Yamada est banquier au Japon.
  5. ワット さん　は　イギリス　の　いしゃ　です。 = M. Watt est médecin anglais.
  6. ワンさん　は　中国ちゅうごく人　の　インジにゃ　です。= M. Wan est ingénieur chinois.
  7. クーパー　は　パリ の 学生がくせい です。Cooper est étudiant parisien.

Donner son âge

  • （あなた は¹）　なん　さい　ですか？ = Quel âge as tu ?
  • わたし　は　３２　さい　です。 = J’ai 32 ans.
  • （お²）とし³　は　いくつ　ですか？ = Quel âge as tu ? (forme plus polie)
  1. On peut omettre la partie entre parenthèses et être plus direct.
  2. お = forme respectueuse qui peut être omise.
  3. 年とし =　année
