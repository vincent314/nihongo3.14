forme 「んです」

Pour donner un complément d'informations. Exemple :

  • 書きます= j'écris (forme polie)
  • 書くんです = j'écris (forme neutre)
  • 書かないんです = je n'écris pas (forme neutre)
  • 書いたんです = j'ai écrit (forme neutre)
  • 書かなかったんです = je n'ai pas écrit (forme neutre)
