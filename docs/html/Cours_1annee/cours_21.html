Cours 21

Grammaire

Présent : V+ます。

exemple : 今日きょう　勉強べんきょうします。 = Aujourd’hui j’étudie

Passé : V+ました。

exemple : 昨日きのう　勉強べんきょうしました。 = Hier j’ai étudié.

Passé négatif : V+ませんでした。

exemple : 昨日きのう　勉強べんきょうしませんでした。 = Hier je n’ai pas étudié.

Dates

Exemple : 2012年ねん　4月がつ　16日にち

  • 1月　から　3月　まで = de janvier à mars
  • 2時　から　4時　まで = de 2 heures à 4 heures
  • いえ　から　えき　まで = de la maison à la gare

Compléments d’objet を et de lieu で (cf. page 65)

  • とけいを買かいます。 = J’achète une montre
  • レストランで食たべます。= Je mange au restaurant.
  • わたしは　イギリスで　ぼうしを　かいました。 = J’ai acheté un chapeau en Angleterre.
  • イタリアで　何なにを　かいましたか？ = Qu’as-tu acheté en Italie ?
  • すてきなさいふをかいました = J’ai acheté un chouette portefeuille.
  • きっさてんで　どんなコーヒーを　のみましたか？ = Quel café as-tu bu au salon de thé ?

Vocabulaire

  • 勉強します = べんきょうします = étudier
  • 起きます = おきます = se lever
  • 寝ます = ねます = se coucher
  • 働きます = はたらきます = travailler
  • 終わります = おわります = finir
  • 休みます = やすみます = se reposer
  • 食べます = たべます = manger
  • バス=bus
  • でんしゃ=train
  • トラム=tramway
  • から=depuis … *まで=jusqu’à …
